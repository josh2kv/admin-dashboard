# 관리자모드 웹페이지

## 1. 소개 및 특징
- 부트스트랩을 활용한 반응형 웹페이지
- 자바스크립트 없이 외관만 구현된 상태
- 입사지원을 위해 지원회사에 맞춰 약간 수정(21.03.27)

## 2. 사용기술
- HTML
- CSS
- BOOTSTRAP

## 3. 학습후기
웹개발 학습 1개월차
- 사용하기 쉬운 부트스트랩을 사용하여 웹페이지 구현에 큰 어려움은 없었으나 구현 난도에 비해 오래걸림
- 빨리 서버와 정보를 주고 받는 법을 배워서 제대로된 작동하는 웹페이지를 완성하고 싶었음
